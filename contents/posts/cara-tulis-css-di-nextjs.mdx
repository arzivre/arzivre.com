# Cara tulis CSS di Nextjs

Nextjs memiliki beberapa cara untuk menulis CSS

- Global CSS
  dengan menulis css di external file

- Components styles dengan Styled JSX

```jsx
<nav>
  <h1>Brand</h1>
  <style jsx>
    {`
      h1 {
        color: blue;
      }
    `}
  </style>
</nav>
```
