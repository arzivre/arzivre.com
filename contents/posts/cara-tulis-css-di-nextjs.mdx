# Cara tulis CSS di Nextjs

Nextjs memiliki beberapa cara untuk menulis CSS

1. Inline style

```jsx
<p style={{ background: 'none', color: '#101113', padding: 0 }}>Text</p>
```

2. Global CSS
   dengan menulis css di external file

3. Components styles dengan Styled JSX

```jsx
<nav>
  <h1>Brand</h1>
  <style jsx>
    {`
      h1 {
        color: blue;
      }
    `}
  </style>
</nav>
```
