# Arzivre.com

Framework: Next.js
Styling: Tailwind CSS
Deployment: Vercel

## Running Locally
This application requires Node.js v16 

```
 git clone https://github.com/arzivre/arzivre.com
 cd arzivre.com
 yarn
 yarn dev
```
